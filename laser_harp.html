Laser Harp

A musical instrument thats puts a spin on a harp by replacing its strings with lasers. It plays a corresponding note when the user touches one of the laser beams.

Description

This was my first real electrical engineering project and I had so much fun learning about all the various components and building it. The laser harp is frameless, meaning the harp does not have receivers at the other end of the laser beam but instead detects the lasers reflection if you touch the beam. This allowed for a more compact design and required no setup.

  •   Github

How it works

It works by shining a one laser onto a galvanometer, a mirror that can rotate back and forth very fast, then it measures if a reflection is detected with a photo resistor. If a reflection is detected a note is played. We were able to achieve multiple beams by moving the galvanometers position therefor moving the laser beams location. Logically it works as follows: The mirror moves to position 1, then waits for a small amount of time. The detector measures the light and determines if a reflection is detected. If a reflection is detected note 1 is played or else nothing happens. Then the mirror moves to position 2 and repeats he same process. This process repeats itself fast enough that a human eye cannot detect it and instead looks like multiple beams. The video on the left shows the laser harp in action for the first time! For this test had four different "strings". The lasers are not visible in clear air so smoke is required to see them. In the video it is difficult to see the lasers becasue we tried to collect smoke by burning a peice of paper. It was not an effeciive method. We later got a fog machine.
